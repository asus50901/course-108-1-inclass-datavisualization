# 7.1.7 CSS

Now, let's explore the maneuver of the CSS elements and commands.
The CSS cmds can be placed inside the "header" or the beginning of the "body".


The below code will set a styling specifically applied to all image objects with dashed border and a aliceblue colored padding border.
Now, we can't actually see the CSS "style" codes, you'll have to open up an element inspector to read the said code.

"img, #myTargetElement, .myBlue" is called CSS selector. It defines which element(s) should the style applied to. (# for id attr, . for class attr)

We must remember that `class` is not unique, while `id` is unique to an object, meaning that `class` tag can be reused but `id` tag must be specified and not able for recycling.



Now, we have another code chunk (obviously, you need the inspector for this invisible code).
We will explain what the code does below.

Now, this is the code for # 7.1.7.1. This is for a CSS file creation and usage.

"img, #myTargetElement, .myBlue { border-style: dashed; padding: 15px 15px; background-color: aliceblue }"

We put this code in a `.css file` and we later add the code below in the `html header` to import the CSS file's effect

"< link rel="stylesheet" type="text/css" href="css/my.css">"
